# escher-cli
CLI for EscherAuth in Ruby

## Requirements
 - Ruby 2.2.3
 - Bundler gem

## Install
When you've met the requirements, cd into the escher-cli directory and:
```
chmod +x ./escher-cli.rb
bundle install
```

## Usage
Simply: `./escher-cli.rb` and follow the instructions.
